## Installation
To get a local copy up and running, follow these simple steps.

1. Clone the repo:
   ```sh
    git clone https://github.com/beiyi1996/resume.git
   ```

2. Install dependencies:
   ```sh
    npm install
   ```

3. Run the app on localhost:
   ```sh
    npm start
   ```

4. Run the tests in watch mode:
   ```sh
    npm test
   ```
